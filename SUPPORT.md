# Support

## Documentation First

Please read these documents before opening issues:
- README.md
- CONTRIBUTING.md
- SECURITY.md

## Where to Ask

- Bug: use Bug report issue template
- Feature idea: use Feature request template
- Translation: use Translation update template
- Legal/compliance: use Legal or compliance notice template

## What to Include

- OS and Python version
- Reproduction steps
- Logs or screenshots where applicable

This helps maintainers respond faster.
