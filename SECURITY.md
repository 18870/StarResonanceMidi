# Security Policy

## Supported Versions

Security fixes are provided for the latest state of the default branch.

## Reporting a Vulnerability

Please do not open a public issue for sensitive vulnerabilities.

Use one of these options:
- GitHub Security Advisories (preferred)
- Directly contact the maintainer: https://github.com/etanolo

When reporting, include:
- Affected file(s) and behavior
- Reproduction steps
- Potential impact
- Suggested mitigation if available

## Response

The maintainer will acknowledge receipt as soon as possible and coordinate a fix/release timeline.
